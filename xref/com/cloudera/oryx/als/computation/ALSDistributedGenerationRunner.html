69  /**
70   * <p>This class will run one generation of the Computation Layer. It is essentially a client of Hadoop
71   * and starts and monitors a series of MapReduce jobs that complete the Computation Layer's work.</p>
72   *
73   * <p>This is an overview of the sequence of MapReduce jobs that can run as part of the ALS pipeline.
74   * While it is one flow, there are several distinct concerns inside of it, some optional. The core
75   * flow is presented first, followed by notes about secondary or optional aspects of the flow. These
76   * are presented in order of execution although in many cases some jobs run in parallel with others.</p>
77   *
78   * <p>In all cases, generation N's data is found at HDFS path {@code /[bucket]/[N]/} (where N is
79   * padded to 5 digits so that lexical order equals numeric order). Locations are relative to this
80   * unless otherwise specified.</p>
81   *
82   * <ul>
83   * <li>"varlong" refers to the representation {@code long}</li>
84   * <li>"varint" refers to {@code int}</li>
85   * <li>"idvalue" refers to {@code IDValue}</li>
86   * <li>"floatarray" refers to {@code float[]}</li>
87   * <li>"floatmap" refers to {@code FastByIDFloatMap}</li>
88   * <li>"idset" refers to {@code FastIDSet}</li>
89   * </ul>
90   *
91   *
92   * <h2>Core Steps</h2>
93   *
94   * <h3>SplitTestStep</h3>
95   *
96   * <p>New input is always found in generation N under {@code inbound/}, in (compressed) CSV text files.
97   * This step splits the data into test and train data (if {@code model.test-set-fraction} is nonzero).
98   * These go into {@code test/} and {@code train/}, respectively, unmodified.</p>
99   *
100  * <h3>MergeNewOldStep</h3>
101  *
102  * <p>Merges all data from last generation (including the last generation's test data)
103  * with new input in the current generation. The last generation's
104  * data exists in generation N-1 under {@code input/} and {@code test/} (because the test data was not part of the
105  * input to the prior generation). It encodes user,item,value tuples as
106  * user-(item,value) key/value pairs, in particular as varlong mapped to idvalue objects.</p>
107  *
108  * <p>This step merges old and new values by adding them. Here is where 'decay' of old data takes place too.</p>
109  *
110  * <p>Output goes to generation N's {@code input/}, and this is what drives all further computation in
111  * the generation.</p>
112  *
113  * <h3>ToUserVectorsStep / ToItemVectorsStep</h3>
114  *
115  * <p>These simply transform the output above into a mapping from user to all item/value pairs as a single
116  * floatmap object -- user vectors if you like -- and likewise for item vectors. The output goes to
117  * {@code userVectors/} and {@code itemVectors/} respectively.</p>
118  *
119  * <h3>CollectKnownItemsStep</h3>
120  *
121  * <p>Outputs user IDs mapped to a set of item IDs that the user ID has interacted with in the data set.
122  * The output is in text format, in {@code knownItems/} and is for consumption by the Serving Layer.</p>
123  *
124  * <h3>InitialYStep</h3>
125  *
126  * <p>This process outputs an initial Y matrix. It will optionally reuse the final {code Y/} output from
127  * last generation, N-1. For any items that have no vector yet (which might be all of them on a fresh run)
128  * it will choose random unit vectors (chosen with kmeans++-style preference for dissimilar vectors).
129  * Output goes to {@code tmp/iterations/0/Y}, and is encoded as varlong mapped to floatarray.</p>
130  *
131  * <h3>PopularUserStep / PopularItemStep</h3>
132  *
133  * <p>Slightly misnamed, as they actually output the set of items that a given user interacts with in the
134  * data, and vice versa for items. The output is encoded as varlong mapped to idset, and the output goes
135  * to {@code popularUsersByItemPartition} and {@code popularItemsByUserPartition} respectively.. This is
136  * used later to cleverly side-load only the part of the X or Y matrix that will actually be used by
137  * a reducer.</p>
138  *
139  * <h3>RowStep</h3>
140  *
141  * <p>The heart of the ALS iteration, and the most important step in the flow. This is the one that is
142  * iterated many times. It will alternately compute X from Y, then Y from X. In the first case, for
143  * iteration i, input is from {@code tmp/iterations/[i-1]/Y} and output goes to
144  * {@code tmp/iterations/[i]/X}. In the second case, it's {@code tmp/iterations/[i]/X} going to
145  * {code tmp/iterations/[i]/Y}. Input and output are rows of a matrix: varlong mapped to floatarray.</p>
146  *
147  * <p>The core ALS computation here is fairly straightforward, but, there is complexity around
148  * side-loading the X or Y matrix, rather than literally joining the matrix on the input, which would
149  * be far too slow. To conserve memory, only keys that are expected to appear in the input to the
150  * particular reducer will be loaded.</p>
151  *
152  * <h3>PublishXStep / PublishYStep</h3>
153  *
154  * <p>Simple map-only jobs that publish the X or Y matrix, respectively, from the final iteration, in a
155  * form that's consumable by the Serving Layer. The output is compressed text format. See the source
156  * code for exact format. It is otherwise the same matrix as computed by the iterations. The output
157  * goes to {@code X/} and {@code Y/} respectively.</p>
158  *
159  * <h2>Recommend Steps</h2>
160  *
161  * <h3>DistributeRecommendWorkStep</h3>
162  *
163  * <p>This first step effectively joins data from {@code X/} (user-feature matrix) and {@code knownItems/},
164  * since both are needed to recommend for a user. It writes the combined values as varlong mapped to
165  * {@code IDAndKnownItemsAndFloatArrayWritable}. The output goes to {@code tmp/distributeRecommend/}.
166  * This is necessary as its own step because the shuffle/combiner will need to be used differently in
167  * the next step.</p>
168  *
169  * <h3>RecommendStep</h3>
170  *
171  * <p>This computes recommendations given output above. The item-features matrix, Y, is also needed.
172  * It is side-loaded cleverly as in {@code RowStep}. Each reducer makes N recommendations per user when
173  * N recommendations are desired. They will be condensed into N overall in the next step. The output
174  * goes to {@code tmp/partialRecommend} and is encoded as varlong (user ID) mapped to a series of idvalue
175  * objects representing top recommended items.</p>
176  *
177  * <h3>CollectRecommendStep</h3>
178  *
179  * <p>This stage simply takes output above and outputs final N recommendations for each user. Output
180  * is conceptually similar, but in text format, and goes to {@code recommend/}.
181  *
182  *
183  * <h2>Similarity Steps</h2>
184  *
185  * <h3>DistributeSimilarWorkStep</h3>
186  *
187  * <p>Quite analogous to {@code DistributeRecommendWorkStep} and {@code RecommendStep} combined. Because
188  * there is no join needed, these two tasks can be accomplished in a single job.</p>
189  *
190  * <h3>SimilarStep</h3>
191  *
192  * <p>Quite analogous to {@code CollectRecommendStep}.</p>
193 
194  * @author Sean Owen
195  */
196 public final class ALSDistributedGenerationRunner extends DistributedGenerationRunner {

200   @Override
201   protected List<DependsOn<Class<? extends JobStep>>> getPreDependencies() {
202     List<DependsOn<Class<? extends JobStep>>> preDeps = Lists.newArrayList();
203     preDeps.add(DependsOn.<Class<? extends JobStep>>first(MergeIDMappingStep.class));
204     preDeps.add(DependsOn.<Class<? extends JobStep>>nextAfterFirst(MergeNewOldStep.class, SplitTestStep.class));
205     preDeps.add(DependsOn.<Class<? extends JobStep>>nextAfterFirst(ToUserVectorsStep.class, MergeNewOldStep.class));
206     preDeps.add(DependsOn.<Class<? extends JobStep>>nextAfterFirst(ToItemVectorsStep.class, MergeNewOldStep.class));
207     preDeps.add(DependsOn.<Class<? extends JobStep>>nextAfterFirst(CollectKnownItemsStep.class,
208                                                                    ToUserVectorsStep.class));
209     preDeps.add(DependsOn.<Class<? extends JobStep>>nextAfterFirst(InitialYStep.class, ToItemVectorsStep.class));
210     preDeps.add(DependsOn.<Class<? extends JobStep>>nextAfterFirst(PopularUserStep.class, ToUserVectorsStep.class));
211     preDeps.add(DependsOn.<Class<? extends JobStep>>nextAfterFirst(PopularItemStep.class, ToItemVectorsStep.class));
212     return preDeps;
213   }
214 
215   @Override
216   protected List<DependsOn<Class<? extends JobStep>>> getIterationDependencies() {
217     List<DependsOn<Class<? extends JobStep>>> iterationsDeps = Lists.newArrayList();
218     iterationsDeps.add(DependsOn.<Class<? extends JobStep>>first(RowStep.class));
219     return iterationsDeps;
220   }

222   @Override
223   protected List<DependsOn<Class<? extends JobStep>>> getPostDependencies() {
224     Config config = ConfigUtils.getDefaultConfig();
225     List<DependsOn<Class<? extends JobStep>>> postDeps = Lists.newArrayList();
226     postDeps.add(DependsOn.<Class<? extends JobStep>>first(PublishXStep.class));
227     postDeps.add(DependsOn.<Class<? extends JobStep>>first(PublishYStep.class));
228     if (config.getBoolean("model.recommend.compute")) {
229       postDeps.add(DependsOn.<Class<? extends JobStep>>nextAfterFirst(RecommendStep.class,
230                                                                       DistributeRecommendWorkStep.class));
231       postDeps.add(DependsOn.<Class<? extends JobStep>>nextAfterFirst(CollectRecommendStep.class,
232                                                                       RecommendStep.class));
233     }
234     if (config.getBoolean("model.item-similarity.compute")) {
235       postDeps.add(DependsOn.<Class<? extends JobStep>>nextAfterFirst(SimilarStep.class,
236                                                                       DistributeSimilarWorkStep.class));
237     }
238     return postDeps;
239   }

254   @Override
255   protected void runOneIteration(int iterationNumber, List<Collection<Class<? extends JobStep>>> iterationSchedule)
256       throws InterruptedException, JobException, IOException {
257     for (Collection<Class<? extends JobStep>> iterationStepClasses : iterationSchedule) {
258       runSchedule(iterationStepClasses, buildConfig(iterationNumber, true)); // X
259     }
260     for (Collection<Class<? extends JobStep>> iterationStepClasses : iterationSchedule) {

265   @Override
266   protected boolean areIterationsDone(int iterationNumber) throws IOException {
267 
268     // First maybe output MAP
269     Store store = Store.get();
270     String iterationsPrefix = Namespaces.getIterationsPrefix(getInstanceDir(), getGenerationID());
271 
272     String mapKey = iterationsPrefix + iterationNumber + "/MAP";
273     if (store.exists(mapKey, true)) {
274       double map;
275       BufferedReader in = store.readFrom(mapKey);
276       try {
277         String line = in.readLine();
278         Preconditions.checkNotNull(line);
279         map = Double.parseDouble(line);
280       } finally {
281         in.close();
282       }
283       log.info("Mean average precision estimate: {}", map);
284     }
285 
286     if (iterationNumber < 2) {
287       log.info("Iterations not done because only on iteration {}", iterationNumber);
288       return false;
289     }
290 
291     Config config = ConfigUtils.getDefaultConfig();
292     int maxIterations = config.getInt("model.iterations.max");
293 
294     if (maxIterations > 0 && iterationNumber >= maxIterations) {
295       log.info("Reached iteration limit");
296       return true;
297     }
298 
299 
300     String previousConvergenceKey = iterationsPrefix + (iterationNumber-1) + "/Yconvergence/";
301     String currentConvergenceKey = iterationsPrefix + iterationNumber + "/Yconvergence/";
302     Preconditions.checkState(store.exists(currentConvergenceKey, false),
303                              "No convergence samples in current iteration");
304     if (!store.exists(previousConvergenceKey, false)) {
305       // Previous iteration was deleted already because next iteration failed to start; continue
306       return false;
307     }
308 
309     LongObjectMap<LongFloatMap> previousEstimates = readUserItemEstimates(previousConvergenceKey);
310     LongObjectMap<LongFloatMap> estimates = readUserItemEstimates(currentConvergenceKey);
311     Preconditions.checkState(estimates.size() == previousEstimates.size(),
312                              "Estimates and previous estimates not the same size: %s vs %s",
313                              estimates.size(), previousEstimates.size());
314 
315     DoubleWeightedMean averageAbsoluteEstimateDiff = new DoubleWeightedMean();
316     for (LongObjectMap.MapEntry<LongFloatMap> entry : estimates.entrySet()) {
317       long userID = entry.getKey();
318       LongFloatMap itemEstimates = entry.getValue();
319       LongFloatMap previousItemEstimates = previousEstimates.get(userID);
320       Preconditions.checkState(itemEstimates.size() == previousItemEstimates.size(),
321                                "Number of estimates doesn't match previous: {} vs {}",
322                                itemEstimates.size(), previousItemEstimates.size());
323       for (LongFloatMap.MapEntry entry2 : itemEstimates.entrySet()) {
324         long itemID = entry2.getKey();
325         float estimate = entry2.getValue();
326         float previousEstimate = previousItemEstimates.get(itemID);
327         // Weight, simplistically, by newValue to emphasize effect of good recommendations.
328         // But that only makes sense where newValue > 0
329         if (estimate > 0.0f) {
330           averageAbsoluteEstimateDiff.increment(FastMath.abs(estimate - previousEstimate), estimate);
331         }
332       }
333     }
334 
335     long numSamples = averageAbsoluteEstimateDiff.getN();
336     double convergenceValue;
337     if (numSamples == 0) {
338       // Fake value to cover corner case
339       convergenceValue = FastMath.pow(2.0, -(iterationNumber + 1));
340       log.info("No samples for convergence; using artificial convergence value: {}", convergenceValue);
341     } else {
342       convergenceValue = averageAbsoluteEstimateDiff.getResult();
343       log.info("Avg absolute difference in estimate vs prior iteration over {} samples: {}",
344                numSamples, convergenceValue);
345       if (!Doubles.isFinite(convergenceValue)) {
346         log.warn("Invalid convergence value, aborting iteration! {}", convergenceValue);
347         return true;
348       }
349     }
350 
351     double convergenceThreshold = config.getDouble("model.iterations.convergence-threshold");
352     if (convergenceValue < convergenceThreshold) {
353       log.info("Converged");
354       return true;
355     }
356     return false;
357   }

359   private static LongObjectMap<LongFloatMap> readUserItemEstimates(String convergenceSamplePrefix)
360       throws IOException {
361     log.info("Reading estimates from {}", convergenceSamplePrefix);
362     LongObjectMap<LongFloatMap> userItemEstimate = new LongObjectMap<LongFloatMap>();
363     Store store = Store.get();
364     for (String prefix : store.list(convergenceSamplePrefix, true)) {
365       for (CharSequence line : new FileLineIterable(store.readFrom(prefix))) {
366         String[] tokens = DelimitedDataUtils.decode(line, ',');
367         long userID = Long.parseLong(tokens[0]);
368         long itemID = Long.parseLong(tokens[1]);
369         float estimate = Float.parseFloat(tokens[2]);
370         LongFloatMap itemEstimate = userItemEstimate.get(userID);
371         if (itemEstimate == null) {
372           itemEstimate = new LongFloatMap();
373           userItemEstimate.put(userID, itemEstimate);
374         }
375         itemEstimate.put(itemID, estimate);
376       }
377     }
378     return userItemEstimate;
379   }

381   @Override
382   protected void doPost() throws IOException {
383     log.info("Loading X and Y to test whether they have sufficient rank");
384 
385     String generationPrefix = Namespaces.getInstanceGenerationPrefix(getInstanceDir(), getGenerationID());
386     String xPrefix = generationPrefix + "X/";
387     String yPrefix = generationPrefix + "Y/";
388 
389     Store store = Store.get();
390     if (!doesXorYHasSufficientRank(xPrefix) || !doesXorYHasSufficientRank(yPrefix)) {
391       log.warn("X or Y does not have sufficient rank; deleting this model and its results");
392       store.recursiveDelete(xPrefix);
393       store.recursiveDelete(yPrefix);
394       store.recursiveDelete(generationPrefix + "recommend/");
395       store.recursiveDelete(generationPrefix + "similarItems/");
396     } else {
397       log.info("X and Y have sufficient rank");
398       File tempModelDescriptionFile = File.createTempFile("model-", ".pmml.gz");
399       tempModelDescriptionFile.deleteOnExit();
400       ALSModelDescription modelDescription = new ALSModelDescription();
401       modelDescription.setKnownItemsPath("knownItems");
402       modelDescription.setXPath("X");
403       modelDescription.setYPath("Y");
404       modelDescription.setIDMappingPath("idMapping");
405       ALSModelDescription.write(tempModelDescriptionFile, modelDescription);
406       store.upload(generationPrefix + "model.pmml.gz", tempModelDescriptionFile, false);
407       IOUtils.deleteRecursively(tempModelDescriptionFile);
408     }
409 
410     super.doPost();
411   }

413   private static boolean doesXorYHasSufficientRank(String xOrYPrefix) throws IOException {
414     double[][] transposeTimesSelf = null;
415     Store store = Store.get();
416     for (String xOrYFilePrefix : store.list(xOrYPrefix, true)) {
417       for (String line : new FileLineIterable(store.readFrom(xOrYFilePrefix))) {
418         int tab = line.indexOf('\t');
419 
420         float[] elements = DataUtils.readFeatureVector(line.substring(tab + 1));
421         int features = elements.length;
422 
423         if (transposeTimesSelf == null) {
424           transposeTimesSelf = new double[features][features];
425         }
426 
427         for (int i = 0; i < features; i++) {
428           double rowFactor = elements[i];
429           double[] transposeTimesSelfRow = transposeTimesSelf[i];
430           for (int j = 0; j < features; j++) {
431             transposeTimesSelfRow[j] += rowFactor * elements[j];
432           }
433         }
434 
435       }
436       
437       // Hacky but efficient and principled: if a portion of the matrix times itself is non-singular,
438       // it's all but impossible that it would be singular when the rest was loaded
439       
440       if (transposeTimesSelf != null && 
441           transposeTimesSelf.length != 0 && 
442           MatrixUtils.isNonSingular(new Array2DRowRealMatrix(transposeTimesSelf))) {
443         return true;
444       } else {
445         log.info("Matrix is not yet proved to be non-singular, continuing to load...");
446       }
447     }
448     return false;
449   }

About Oryx

The Oryx open source project provides simple, real-time large-scale machine learning infrastructure.

This project has declared the following modules:

Oryx Alternating Least Squares Computation ALS model computation

Oryx K-Means Serving K-Means model serving

Oryx Random Decision Forests Common Code common to random decision forest model serving and computation
